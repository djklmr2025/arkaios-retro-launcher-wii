# ROMs locales

Esta carpeta existe solo como guia para uso local.

No se incluyen ROMs comerciales en este repositorio. Si vas a usar el patcher
web de ARKAIOS, selecciona desde tu computadora una copia base compatible y un
parche BPS legal/distribuible.

Para `Super Mario World 2025 (English).bps`, la ROM base esperada por el parche
detectado es:

- Archivo usual: `Super Mario World (USA).sfc`
- Tamano: `524,288 bytes`
- CRC32: `B19ED489`

Si el archivo local del usuario no coincide con ese tamano y CRC32, el patcher
del navegador detiene el proceso y no genera una ROM incorrecta.

Los archivos colocados en esta carpeta quedan ignorados por Git salvo este
`README.md`.
